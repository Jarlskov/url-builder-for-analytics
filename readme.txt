== Description ==
The plugins allows you to create google analytics tracking urls directly from your post editor.

== Installation ==
= From your WordPress dashboard =
1. Visit 'Plugins > Add New'
2. Search for 'url builder for analytics'
3. Active URL Builder for Analytics from your Plugins page.

= From WordPress.org =
1. Download Google Analytics URL Builder
2. Unzip the downloaded zip file
3. Upload the url-builder-for-analytics directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
4. Active URL Builder for Analytics from your Plugins page.
